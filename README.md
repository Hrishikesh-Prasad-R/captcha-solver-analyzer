Set-Content README.md "# CAPTCHA Solver & Analyzer

## Overview
This project uploads various types of CAPTCHA images (text, object, math), attempts to solve them using machine learning/AI models, and analyzes their resistance to neural network-based bots.

## Features
- Supports multiple CAPTCHA types
- Image preprocessing
- Model training for CAPTCHA solving
- Security analysis of CAPTCHA strength

## Tech Stack
Python, Flask, OpenCV, PyTesseract, YOLOv5, PyTorch

## Setup
\`\`\`bash
git clone https://github.com/Hrishikesh-Prasad-R/captcha-solver-analyzer.git
cd captcha-solver-analyzer
pip install -r requirements.txt
